# url_shortener-go
A URL shortener. 
Frontend: React.js, Material-UI
Backend: gofiber

## Quick Start

- To build and run Backend
  - `docker build -t <tag name> ./api && docker run <tag name>`

- To try out frontend with development server.
  - `npm install`
  - `npm start`
  - checkout http://localhost:3000/
